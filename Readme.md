# Домашнее задание по курсу «Базы данных» (SQL)

## Описание проекта

Проект представляет собой выполнение домашнего задания по дисциплине «Базы данных».  
В рамках работы решено **13 практических задач** по четырём тематическим базам данных:

1. **Транспортные средства** (Vehicle, Car, Motorcycle, Bicycle) – 2 задачи.
2. **Автомобильные гонки** (Classes, Cars, Races, Results) – 5 задач.
3. **Бронирование отелей** (Hotel, Room, Customer, Booking) – 3 задачи.
4. **Структура организации** (Departments, Roles, Employees, Projects, Tasks) – 3 задачи.

Все задания выполнены на **SQL (диалект MySQL)** с использованием сложных конструкций:  
`JOIN`, `UNION`, `WITH` (CTE), `RECURSIVE`, агрегатные функции, оконные функции, группировка, конкатенация строк (`GROUP_CONCAT`) и условная логика.
